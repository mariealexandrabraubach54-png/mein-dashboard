# 🌸 Mein Dashboard

Persönliches Dashboard von Marie — Gedanken festhalten (auch per Sprache), in Kategorien
und Unterkategorien sortieren, plus Kalender mit Tag-/Wochen-/Monats-/Jahresansicht.

**App öffnen:** https://mariealexandrabraubach54-png.github.io/mein-dashboard/

Die Daten werden über das private Repository `mein-dashboard-daten` zwischen den Geräten
synchronisiert. Dafür einmal pro Gerät oben auf „☁️ Sync einrichten" tippen und den
GitHub-Token einfügen.
